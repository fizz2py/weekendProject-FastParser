# FastParser

## Overview

FastParser is a C++ project that reads data from a CSV file, parses each row into custom `Person` objects, sorts the records using a custom comparator, and writes the sorted results to a new CSV file.

## Features

- Read CSV files using File I/O
- Parse data into custom C++ objects
- Sort records by salary using a custom comparator
- Efficient insertion using Move Semantics
- Exception handling for file operations
- Output processed data to a new CSV file
- CMake build support
